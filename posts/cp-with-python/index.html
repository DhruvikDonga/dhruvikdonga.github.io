Approach for solving a problemüñã

  • A problem statement mainly revolves around :- Problem , Input , Ouput and constraints . The constraints are 2nd most important 1st is understanding the question .
  • There are 2 main solutions to a problem Brute Force or using an algorithm which fits the constraints and can give full points.
  • To get full points you need to pass all test cases which means you need to manage time constraint which can be 1sec or depending on question.
  • Let’s understand some basic time complexity‚è≥:-

A problem statement will have constraints which can be 0<N<10^6 . Breaking it down we need to understand which algorithm suits this problem the most .
The time is limited 1 sec and N^2 means 10^12 . Now to be clear online compilers of contests generally considers 10^8 to 10^10 range loops under 1sec time my estimate.
So your bruteforce algorithm O(n^2) will not help then let us think for O(NlogN) it is under time range.
This judgment is needed in ACM-style 3hours contests where you need to move fast to score more.
